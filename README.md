📌 Descrição

Este projeto é uma aplicação web simples para exibir informações climáticas em tempo real. Foi desenvolvido como forma de praticar HTML e o consumo de APIs externas, utilizando a OpenWeatherMap.

✨ Funcionalidades

Exibe a temperatura atual da cidade informada.

Mostra a condição climática atual (ex: “Ensolarado”, “Nublado”).

Apresenta um ícone representando o clima.

Possui um formulário para inserção do nome da cidade.

Design responsivo com estilização em tons de verde claro.

🛠️ Tecnologias Utilizadas

HTML – Estrutura da aplicação

CSS – Estilização e layout responsivo

JavaScript – Lógica da aplicação e consumo da API

jQuery – Manipulação do DOM e requisições AJAX

OpenWeatherMap API – Fornecimento dos dados climáticos

🚀 Como Utilizar

Abra o arquivo index.html em um navegador.

Digite o nome da cidade desejada no formulário.

Clique no botão "Buscar Clima" (ou equivalente).

O site exibirá:

Temperatura atual

Condição climática

Ícone correspondente ao clima da cidade informada

📂 Estrutura do Projeto

index.html – Arquivo principal contendo a estrutura da página, formulário e área de exibição dos resultados.

style.css – Arquivo responsável pela estilização da interface.

script.js – Arquivo JavaScript com a lógica para requisição da API e exibição dos dados na tela.

🙌 Créditos

OpenWeatherMap API – Fornecimento dos dados meteorológicos.

jQuery – Simplificação da manipulação do DOM e requisições AJAX.
